# 데이터 구조와 알고리즘

> **Swift 로 작성해 보는 데이터 구조와 알고리즘 문제**
>
> 공부한 내용을 정리하고 기록하기 위한 저장소 입니다.

<br/>

## ✏️ Rule

1. 충분한 시간을 들여 문제를 분석합니다.(30m~1h)
2. 테스트 코드를 작성 합니다.
3. 분석한 내용을 기반으로 코드를 구현해 봅니다.(30m)

```
구현이 잘 안된다면 검색을 합니다.
잘 만들어진 코드를 참고 합시다.🧐
```
